# Zanieczyszczenie powietrza vs. nowotwory
Repozytorium przedstawiające kod strony z projektem zrealizowanym za pomocą biblioteki D3.
Zawiera mapę województw w przedstawioną emisją zanieczyszczeń powietrze, wykres punktowy ze stosunkiem zgonów spowodowanych nowotworami do poziomu zanieczyszczeń powietrza oraz wykres słupkowy wskazujący tendencję poziomu zanieczyszczeń powietrza.

Zapraszam do przeglądania kodu oraz do odwiedzania strony https://michaloruba.pl
